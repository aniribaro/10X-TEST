## What's Included

| File | Purpose |
|------|---------|
| `CUSTOMIZATION.md` | Complete white-label customization guide |
| `branding.json` | Configuration file for colors, logo, name |
| `assets/` | Placeholder for custom logo and images |

## How to Use

1. **Edit `branding.json`** with your company's:
   - App name
   - Logo URL
   - Primary colors
   - Tagline
   - Contact email

2. **Follow `CUSTOMIZATION.md`** for step-by-step instructions on:
   - Replacing the logo
   - Changing color themes
   - Updating meta tags
   - Custom domain setup

3. **Deploy** your white-labeled version
